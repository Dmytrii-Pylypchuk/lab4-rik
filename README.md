# Text Editor (Android)

Мобільний застосунок **“Текстовий редактор”** для ОС **Android**, розроблений у середовищі **Android Studio** в рамках лабораторної роботи з дисципліни **“Розробка інтерфейсу користувача”**.

## Опис застосунку

Застосунок реалізує базовий функціонал текстового редактора та дозволяє:

- створювати новий текстовий файл;
- відкривати існуючий текстовий файл із пам’яті пристрою;
- редагувати текст;
- зберігати зміни у файл;
- створювати новий файл та зберігати його у вибране місце;
- отримувати попередження про незбережені зміни перед відкриттям іншого файлу, створенням нового файлу або виходом із програми.

Інтерфейс користувача реалізований відповідно до **принципів якості інтерфейсу** та оформлений у стилі **Material Design**.

---

## Функціональні можливості

### Основні функції:
- **New File** — створення нового документа;
- **Open File** — відкриття існуючого текстового файлу через системний File Picker;
- **Save File** — збереження змін:
  - якщо файл уже відкритий — виконується перезапис;
  - якщо файл новий — відкривається діалог вибору місця збереження;
- **Exit** — завершення роботи застосунку.

### Додаткові можливості:
- **Перевірка незбережених змін** перед:
  - створенням нового файлу;
  - відкриттям іншого файлу;
  - виходом із програми;
- **Діалогове вікно підтвердження**:
  - Save
  - Discard
  - Cancel
- **Лічильник символів** у тексті;
- **Floating Action Button** для швидкого збереження;
- **Сучасна обробка кнопки Back** через `OnBackPressedDispatcher`.

---

## Використані технології

- **Java**
- **Android Studio**
- **Android SDK**
- **Material Design Components**
- **AndroidX**
- **File Picker API**
  - `Intent.ACTION_OPEN_DOCUMENT`
  - `Intent.ACTION_CREATE_DOCUMENT`
